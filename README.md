# Mario Artist Paint Studio Decompilation
(WIP Readme)

Required Base ROM File:  
`dmpj.d64` - MD5: `3546bc6d93928a28285a91478bc41b68`

Convert Mario Artist Paint Studio NDD dump to D64 using [https://github.com/LuigiBlood/leo64dd_python](https://github.com/LuigiBlood/leo64dd_python).
The RAM Area should be only `0xFF`.

Use `make setup` to do the stuff.
